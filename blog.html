The Document Object Model (DOM) is a programming interface for web documents. It defines the logical structure of documents and the way of a document is accessed and manipulate.

There are 5 ways in which we can select HTML elements using DOM. The most common way to access an HTML element is to use the id as getElementById() method. We can also use mothod like getElementsByClass(), getElementsByTag(), querySelector() and querySelectorAll().

Event bubbles are a concept in the DOM. It occurs when an element receives an event and increments its parent and ancestor elements in the DOM tree until that event reaches the parent element.

A callback function is a function passed into another function as an argument.
